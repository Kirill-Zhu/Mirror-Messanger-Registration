# **ТЕХ Документация**
-----------
-В качестве прототипа я взля пример-проект Mirror "Tanks"

-Чтобы подписаться на сообщения о присоединении пользователя необходимо нажать кнопку **Register**(Слева)

-Чтобы отправить тестовоое сообщение всем подписанным на CustomMessage также пожно нажать **Send**(Справа)
